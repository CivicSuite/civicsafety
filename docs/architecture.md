# CivicSafety Architecture

CivicSafety v0.1.0 is a deterministic FastAPI module over CivicCore. It provides non-CJIS public-safety administrative support while explicitly avoiding CAD/RMS integration, dispatch, enforcement, investigations, evidence workflows, and other high-risk decision surfaces.

![CivicSafety architecture](architecture-civicsafety.svg)

## Shipped

- Non-CJIS policy/SOP answer drafts with citations.
- Training checklist scaffolds for supervisor review.
- PIO-reviewed public-information drafts.
- Aggregate public-statistics summaries.
- Public sample UI and release gates.

## Not Shipped

- CJI ingestion or exposure.
- CAD/RMS integration.
- Dispatch, enforcement, investigations, or evidence workflows.
- Legal advice.
- Live public-safety connectors.
- Live LLM calls.
